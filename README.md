# Smart Neural Digital Twin

This is a prototype of a **Smart Neural Digital Twin for oilfield safety**. It connects with existing sensors, detects anomalies, and provides predictive insights using machine learning.

## Features
- Real-time sensor data dashboard with interactive visualization
- Anomaly detection using Isolation Forest algorithm
- Intervention recommendations based on live data analytics
- Easy-to-use web interface powered by Streamlit

## Requirements
- Python 3.8+
- Streamlit
- scikit-learn
- pandas
- numpy

## Installation
```bash
git clone https://github.com/rrakanmarri1/Smart-neural-digital-twin.git
cd Smart-neural-digital-twin
pip install -r requirements.txt
